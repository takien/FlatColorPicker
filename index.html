Flat color picker v0.2

Flat color picker for Bootstrap.

  • Follow @cektkp
  • Tweet

Features

  • Flat color
  • Responsive
  • Displays recent selection
  • Color search/filter
  • Displays color name, HEX code and RGB
  • Lightweight
